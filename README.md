# Presale Smart Contract Vyper Edition

Vyper `0.4.0` is used in the project.

! Given project was migrated from hardhat to foundry.

## Get Started

```shell
bun install
```

```shell
bun hardhat compile
```

```shell
bun hardhat test
```

## Misc

Format your Solidity code
```shell
bun run solhint
```
